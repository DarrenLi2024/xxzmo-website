# 闲心子墨 管理后台升级改造计划

> 基于 vue-manage-system (18K★) 设计模式 + tailadmin-nextjs + shadcn-ui skills

---

## 一、改造总览

```
当前状态                    目标状态
────────────────────────    ────────────────────────
4 张统计卡片                Dashboard + ECharts 趋势图
简单表格列表                专业表格(批量操作/列设置/行内搜索/导出)
alert() 错误提示             Toast 通知系统
无标签页                    多标签页编辑工作台
单页表单                    分步表单 + 实时校验
只读配图库                  配图上传/管理/关联
基础标签管理                标签合并/重命名/批量操作
无操作日志                  最近操作时间线
静态页面                    骨架屏 + 空状态插画
```

## 二、组件体系升级

### 2.1 新增通用组件

| 组件 | 参考来源 | 功能 |
|------|---------|------|
| `DataTable` | vue-manage-system `table-custom` | 批量选择、列显隐、行内搜索、排序、分页 |
| `TableToolbar` | vue-manage-system `table-toolbar` | 批量删除、刷新、列设置、导出 |
| `SearchForm` | vue-manage-system `table-search` | 动态搜索表单(input/select/date) |
| `Toast` | shadcn-ui `toast` | 操作反馈通知(成功/失败/警告) |
| `StatCard` | tailadmin-nextjs | 统计卡片(图标+数值+趋势箭头) |
| `Skeleton` | 自研 | 统一骨架屏组件 |
| `EmptyState` | 自研 | 统一空状态插画+引导文字 |
| `ConfirmDialog` | shadcn-ui `dialog` | 统一确认弹窗(替代 confirm()) |
| `TabsManager` | vue-manage-system `tabs` | 多标签页工作台 |
| `ChartCard` | tailadmin-nextjs | ECharts 图表卡片容器 |

### 2.2 已有组件重构

| 组件 | 当前问题 | 改造方案 |
|------|---------|---------|
| `AdminSidebar` | 硬编码 items，与常量重复 | 从 `ADMIN_NAV_ITEMS` 统一导入 |
| `AdminLayout` | 无标签页、无面包屑 | 添加 TabsManager + Breadcrumb |
| `AdminArticleList` | 无批量操作 | 集成 DataTable + TableToolbar |
| `ArticleForm` | 无实时校验、无分步 | 添加字段级校验 + 分步(元数据→正文→AI补全) |
| `FilterBar` | 独立搜索，不与表格联动 | 集成到 SearchForm + DataTable |

## 四、交互体验升级

### 4.1 Toast 通知系统
- 替代所有 `alert()` 调用
- 类型: success / error / warning / info
- 位置: 右上角
- 自动消失 3s
- 支持手动关闭

### 4.2 多标签页工作台
- 侧边栏点击 → 打开新标签
- 标签可关闭、可切换
- "关闭其他"/"关闭所有" 右键菜单
- 标签状态保持 (表单内容不丢失)

### 4.3 骨架屏统一
- 替换所有 ad-hoc `animate-pulse` div
- `<Skeleton variant="table-row" />` 
- `<Skeleton variant="card" />`
- `<Skeleton variant="form" />`

### 4.4 空状态设计
- 替换纯文字空状态
- 插图 + 引导文字 + 行动按钮
- 例如: "暂无文章" → "🍂 山房主人尚未挥毫" + [新建第一篇文章]

### 4.5 确认弹窗统一
- 替代所有 `confirm()` 原生弹窗
- `<ConfirmDialog title="确认删除" description="此操作不可恢复" />`
- 支持危险操作红色按钮

## 六、技术选型

| 功能 | 方案 | 理由 |
|------|------|------|
| 图表 | ECharts (echarts-for-react) | 中文友好，文档丰富 |
| Markdown | react-markdown + rehype | 轻量，SSR 友好 |
| Toast | shadcn/ui toast (已有 radix) | 项目已用 radix |
| 图标 | lucide-react (已有) | 保持一致性 |
| 表单校验 | 自定义 validate 模块 (已有) | 无需新增依赖 |
